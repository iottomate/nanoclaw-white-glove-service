# Service Overview

A comprehensive look at the NanoClaw White-Glove Service offering.

## Executive Summary

NanoClaw White-Glove Service delivers professional installation and support for personal AI assistants built on Claude AI. Unlike ChatGPT or other cloud-based solutions, NanoClaw runs on customer hardware, ensuring complete privacy and data control while integrating seamlessly with WhatsApp.

## What Makes This Service Unique

### 1. Privacy-First Architecture
- All data stays on customer hardware (Mac mini or local server)
- No cloud processing beyond Anthropic's Claude API
- Container isolation for security
- Customer owns their data completely

### 2. WhatsApp Native Integration
- Works in the app customers already use daily
- No new application to learn or install
- Familiar interface with AI superpowers
- Support for other platforms (Telegram, Discord, Slack)

### 3. Context Isolation
- Multiple WhatsApp groups = multiple contexts
- Finance group for market analysis
- Projects group for task management
- Personal group for life management
- Each group has separate memory and files

### 4. White-Glove Experience
- Complete hardware procurement (if needed)
- Full installation and configuration
- Custom group setup per customer needs
- Comprehensive training and documentation
- 30-day support period included

## Technology Stack

### Hardware Options
1. **Mac mini (Recommended)**
   - M1/M2/M3 chip (8-32GB RAM)
   - Silent, energy-efficient
   - Excellent for 24/7 operation
   - Apple Container support

2. **Raspberry Pi 4**
   - Budget option ($100)
   - 4-8GB RAM
   - Low power consumption
   - Docker container support

3. **Cloud VPS**
   - DigitalOcean, Linode, AWS
   - $20-40/month
   - No hardware maintenance
   - Trade-off: Data in cloud

### Software Stack
- **NanoClaw:** Open source (MIT license)
- **Node.js:** JavaScript runtime
- **Baileys:** WhatsApp Web API library
- **SQLite:** Message history database
- **Apple Container / Docker:** Security isolation

### AI & Integration
- **Claude AI:** Anthropic's best-in-class LLM
- **Claude Code:** API access tier
- **WhatsApp:** Primary messaging platform
- **Alternative:** Telegram, Discord, Slack, Signal
